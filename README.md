# Python Assignment Part 2 – Data Structures

## Description

This repository contains the solution for Part 2: Data Structures.
The project is based on a Restaurant Menu and Order Management System implemented using Python.

---

## File Included

* part2_order_system.py – Main script containing all tasks

---

## Tasks Implemented

### Task 1: Explore the Menu

* Displayed menu grouped by category (Starters, Mains, Desserts)
* Counted total menu items
* Counted available items
* Identified most expensive item
* Listed items priced below 150

---

### Task 2: Cart Operations

* Created cart using list of dictionaries
* Added items with quantity handling
* Updated quantity for duplicate items
* Handled unavailable and invalid items
* Removed items from cart
* Generated order summary with subtotal, GST (5%), and total

---

### Task 3: Inventory Tracker

* Used deep copy to create backup of inventory
* Demonstrated that backup remains unchanged
* Deducted stock after order placement
* Displayed reorder alerts for low stock items

---

### Task 4: Sales Log Analysis

* Calculated daily revenue
* Identified best-selling day
* Found most ordered item
* Added new sales data and updated analysis
* Printed all orders using enumerate
